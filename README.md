# hypodrive

Run [hyposcript](https://github.com/sure-thing/hyposcript) projects (almost) natively.

```bash
npm i hypodrive -D
```

## Usage

```bash
npx hypodrive index.js
```

Alternatively, you can run node with a prequire flag:

```bash
node -r hypodrive index.js
```
